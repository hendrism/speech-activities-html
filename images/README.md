# Images Directory

This folder contains organized images for use across all speech therapy activities.

## Folder Organization

- **food/** - Food items (fruits, vegetables, meals, snacks, drinks)
- **animals/** - Animals (pets, farm animals, wild animals, birds, insects)
- **objects/** - Common objects (toys, tools, household items, school supplies)
- **seasons/** - Seasonal items (fall leaves, pumpkins, snowflakes, flowers, etc.)
- **emotions/** - Emotion faces and expressions
- **people/** - People doing various activities
- **places/** - Locations (school, park, home, store, etc.)
- **activities/** - Actions (running, eating, playing, reading, etc.)

## Quick Start

1. **Add images**: Save your images in the appropriate folder
2. **Name clearly**: Use descriptive names like `red-apple.png` or `happy-face.jpg`
3. **Use in activities**: Reference with `images/food/apple.png`

## Example Usage

```html
<img src="images/food/apple.png" alt="Apple" width="200">
```

See **IMAGE_GUIDE.md** in the parent directory for complete documentation and examples.
